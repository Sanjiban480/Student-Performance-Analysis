# Student Performance Analysis Dashboard
## Dashboard Preview

![Dashboard Preview](dashboard/screenshots/dashboard_overview.png)

## Project Overview
This project analyzes student academic performance using Power BI. The dashboard provides insights into student demographics, study habits, attendance patterns, and academic performance.

## Tools Used
- Power BI
- GitHub

## Dataset
Student Performance Dataset containing:
- Gender
- Study Time
- Absences
- Parent Education
- School Information
- Final Grades

## Key Insights
- Students with higher study time generally achieved better grades.
- Female students slightly outperformed male students.
- Parent education showed a positive relationship with student performance.
- Excessive absences negatively impacted final grades.

## Dashboard Features
- KPI Cards
  - Total Students
  - Average Final Grade
  - Average Study Time
  - Average Absences

- Interactive Filter
  - Gender Filter

- Visualizations
  - Gender Distribution
  - Average Grade by Gender
  - Average Grade by Study Time
  - Average Grade by Mother's Education
  - Average Grade by School
  - Grade vs Absences
